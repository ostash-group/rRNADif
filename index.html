Make 16S great again

Get Started

Compare 16S variability

rRNADif will automatically annotate 16S in your genome, compute their variability and compare it to the precomputed database of 21000+ fully sequenced Bacteria genomes (or it's subset).

Build species phylogenetic trees

rRNADif lets you, using the computed variability, see to what extend topology of the phylogenetic tree can be altered. Also it is using comperhensive data from only complete genomes, leading to better species/genus-level classification of genomes based on their rRNAs

Build custom database

Not happy with provided data? No worries! Built you own databases using csv file from GenBank or locally stored genome sequences.
