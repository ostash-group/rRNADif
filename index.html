Make 16S great again

Get Started

Compare 16S variability

rRNADif will automatically annotate 16S in your genome and compare them to precomputed database of 21000+ fully sequenced Bacteria genomes. (or it's subset)

Build species phylogenetic trees

rRNADif lets you not only to measure degree of heterogenuity of 16S rRNA, but also use that information to see to what extend topology of the phylogenetic tree can be altered.

Build custom database

Not happy with provided data? No worries! Built you own databases using csv file from GenBank or locally stored genome sequences.
